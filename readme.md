## Installation

- git clone https://github.com/Siiriirat/Gallery.git
- cd Gallery
- Create a database and inform ".env"
  <p>DB_DATABASE=gallery_db</p>

  ->>DB_USERNAME=root
  ->>DB_PASSWORD=""
- Open command
  ->>php artisan migrate
  ->>php artisan db:seed
  ->>php artisan serve
